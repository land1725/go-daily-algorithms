# Day 03: 有效的括号匹配

## 问题描述

给定一个只包含字符 `'('`, `')'`, `'{'`, `'}'`, `'['`, `']'` 的字符串 `s`，判断该字符串是否有效。有效字符串需满足：

1. 左括号必须用相同类型的右括号闭合
2. 左括号必须以正确的顺序闭合

## 解决方案

### 方法思路

使用**栈**数据结构进行括号匹配检查：

1. 初始化一个空栈和括号映射表（右括号到左括号的映射）
2. 遍历字符串中的每个字符：

- 遇到左括号时入栈
- 遇到右括号时检查是否与栈顶的左括号匹配

3. 最后检查栈是否为空

### 复杂度分析

- **时间复杂度**：O(n)，只需一次完整遍历字符串
- **空间复杂度**：O(n)，最坏情况下需要存储所有左括号
